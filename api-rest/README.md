# Koopers REST API

## Backend

Python 3.8.5

## Installation

This only needs to be run the first time you're going to work on the project:

```bash
virtualenv .venv
```

## Usage

```bash
source .venv/bin/activate
pip install -r requirements.txt
python manage.py runserver 0.0.0.0:8000
```
